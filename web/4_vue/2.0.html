# 如何重写数组方法

首先在数据defineProperty监控阶段将Observer实例保存到数组对象的__ob__ (opens new window)属性中。

然后通过重写数组的methodsToPatch (opens new window)这些方法。 当用户调用这些被重写的方法时，先通过调用原先的方法拿到结果。然后对结果添加observeArray (opens new window)响应式监控。

最后调用ob.dep.notify() (opens new window)通知视图更新。
