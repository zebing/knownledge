# Terminal 控制台设置代理

  1. 开启代理

  2. 添加 Terminal proxy 脚本。 在～/.zshrc文件中，加入以下脚本

# proxy list
alias proxy="export all_proxy=socks5://127.0.0.1:7890"
alias unproxy="unset all_proxy"

然后执行
